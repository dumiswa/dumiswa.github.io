About

This project contains three university assignments from my 3D Rendering course. I built it using C++ and modern OpenGL within the MGE (Micro Game Engine) framework. Each assignment explores advanced rendering concepts such as the Phong lighting model and terrain rendering using heightmaps and splat maps.

Project Info

  • Role: Engineer
  • Team Size: 1
  • Duration: 4 weeks
  • Framework: C++ with OpenGL (MGE)

ADS Lighting Model

This assignment implements the Ambient, Diffuse, and Specular components alongside with attenuation of the Phong lighting model. Directional and point lights are supported, and multiple material types like ColorMaterial, TextureMaterial, and LightMaterial are introduced. Per-fragment lighting is calculated in the shader, with adjustable light properties and specular strength.

Terrain Rendering

This project loads and renders a high-resolution terrain (8196x8196) using a grayscale heightmap. The vertex shader performs displacement, and a splat map blends multiple textures in the fragment shader. Terrain resolution and texture scaling are configurable, and lighting from Assignment 3 is reused for realism.

Highlights

  • Phong ADS lighting model with multiple materials
  • Real-time terrain deformation using vertex shaders
  • Splat map texture blending
  • Per-fragment lighting with adjustable parameters
  • Modular shader utility system
